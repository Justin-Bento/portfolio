After starting at a computer day in and day out. I realized I had mental fatigue with school and being creative, I felt burnted out. So during my break I decided to step away from my laptop and pick up and hold hobby. It was great I took a long deserved break after completing six classes in one semester. However, when the next semester started I realized I had to develop a portfolio website as I was about to graduate this year.

I wanted to create it from scratch. But, I stoped coding for a long time and I did not know where to start. I got nervous and thinking I had to learn everything again, thankfully I was watching developers battle eachother on with css and I thought that would be a great way to get back into website development.

## The Problem?

I had to re-teach myself the basics of website development, and learn Reactjs from scratch while developing my portfolio website, for a school assignment.

## What Did I Decide To Do?

To Re-teach mysellf the basics of website development I decide to use cssbattle.dev. to help me figure out where I am with CSS, and use freecodecamp to help me get the basics down. Learning code through challanges was a fun and exciting way to test mysellf. My rules for this project were simple;

Set a time-limit of 25min.

Only look at CSS documentation 3x.

**Disclaimer:** CSSBattle should not be used to learn CSS or be used in real projects. I used this website to help me get back into Website Development.

## Should You Use CSSBattle?

CSSBattle is a great tool to get you back into CSS, espically if you like to test your skills. It taught me how I should go about reading documentation, alongside some me new CSS tips and tricks.

However, I would not recommend it to someone starting out. As there are weird convections being used to reduce css and it may not be great for learning CSS.
